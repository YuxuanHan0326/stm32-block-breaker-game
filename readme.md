# STM32 Based Handheld Brick Breaker Game
***Note: This is an undergraduate coursework. Permission is obtained from the module lead to upload the project files to GitHub.***

## Introduction
This is a simple brick breaker game, made up of a NUCLEO-F401RE board, a joystick and a [PIMORONI 11x7 LED Matrix Breakout](https://shop.pimoroni.com/products/11x7-led-matrix-breakout?variant=21791690752083) (Powered by an IS31FL3731 driver chip).
